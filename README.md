# 🔐 AI-Powered Cloud Security & Incident Intelligence Platform

- Real-time EC2 monitoring
- CloudTrail security event tracking
- SNS security alerts
- EventBridge automation
- AI-generated threat analysis
- Bedrock GenAI remediation suggestions
- Streamlit security dashboard
